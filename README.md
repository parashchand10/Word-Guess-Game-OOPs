# Word-Guess-Game-OOPs
A Word Guess Game built using Python and Object-Oriented Programming principles. This project demonstrates core OOP concepts like classes, methods, loops, and condition handling. It provides an interactive command-line experience where users guess letters to complete a hidden word. Ideal for beginners to understand real-world OOP implementation in Python.


### Features
1. Random word selection
2. Limited attempts system
3. Letter-by-letter guessing
4. Win/Lose condition handling
5.  Clean and structured OOP implementation

### OOP Concepts Used

1. Class & Object Creation
2. Encapsulation
3. Constructor (__init__)
4. Method Implementation
5. Game Logic Separation

### Learning Outcome

1. Practical understanding of OOP concepts
2. Improved logical thinking
3. Experience in building interactive console applications
